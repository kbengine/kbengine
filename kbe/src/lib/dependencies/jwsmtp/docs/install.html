Compile & Install jwSMTP

Download jwSMTP
Unpack the archive. 
tar -xvzf jwsmtp-<version>
cd jwsmtp-<version>
./configure
make
make install
where <version> is the latest version of jwSMTP without the <> symbols.
This compiles and installs the library.
